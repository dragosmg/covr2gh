# Estimate the length of the value text

Estimate the length of the value text

## Usage

``` r
estimate_text_length_value(badge_value, text_length_label = 50)
```

## Arguments

- badge_value:

  (`badge_value`) the output of
  [`badge_value()`](https://dragosmg.github.io/covr2gh/reference/badge_value.md)

- text_length_label:

  (numeric) the length of the label text. Defaults to 50 pixels.

## Value

numeric representing the length of the label text
