# Estimate the width of the value box

Estimate the width of the value box

## Usage

``` r
estimate_width_value(badge_value, width_label = 60)
```

## Arguments

- badge_value:

  (`badge_value`) the output of
  [`badge_value()`](https://dragosmg.github.io/covr2gh/reference/badge_value.md)

- width_label:

  (numeric) the width of the label box. Defaults to 60 pixels.

## Value

numeric representing the with of the box in which the value will appear
